# projeto_final_topicos_especial_programacao_2025.2
Repositório da disciplina Tópicos Especiais em Programação, contendo a implementação de um pipeline completo de Processamento de Linguagem Natural (PLN) aplicado ao dataset SMS Spam Collection, incluindo pré-processamento, vetorização (BOW e TF-IDF), treinamento de modelos, avaliação de métricas e interpretação com LIME e SHAP.
